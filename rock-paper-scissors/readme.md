# 🎮 Rock Paper Scissors

A modern, responsive Rock Paper Scissors game built with **HTML, CSS, and JavaScript** as part of **The Odin Project** Foundations curriculum. This project demonstrates DOM manipulation, event handling, game logic, and responsive UI development without using any external libraries or frameworks.

---

## ✨ Features

- 🎯 Interactive button-based gameplay
- 🤖 Random computer opponent
- 🔢 5-round game system
- 📊 Live scoreboard (Human, Computer, and Tie)
- 📝 Current round results
- 🏆 Automatic winner announcement
- 🔄 Play Again functionality
- 📱 Responsive design for desktop and mobile
- 🎨 Modern SaaS-inspired user interface

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
