# Landing Page

A responsive-inspired business landing page built as part of **The Odin Project Foundations** curriculum. This project focuses on practicing HTML structure, CSS styling, Flexbox layouts, and page organization by recreating a professional landing page design.

## Features

* Hero section with navigation
* About section with image cards
* Customer testimonial section
* Call-to-action section
* Footer
* Built using only HTML and CSS

## Technologies Used

* HTML5
* CSS3 (Flexbox)

## What I Learned

* Structuring web pages with semantic HTML
* Building layouts using Flexbox
* Applying the CSS Box Model
* Organizing CSS for readability
* Creating reusable sections and consistent spacing

This project was completed as part of my journey through **The Odin Project**.
